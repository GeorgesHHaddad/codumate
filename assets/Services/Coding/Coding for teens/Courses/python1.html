Course Curriculum - Python 1

Welcome to the Introduction to Python Programming course! This 8-week course will help you learn Python fundamentals and build hands-on projects as you progress. Perfect for teens 14+, this course will enhance problem-solving skills and provide an introduction to coding.

Week 1: Introduction to Python and Programming Basics

    Lesson 1: Introduction to Python
    Lesson 2: Python Syntax and Basic Concepts

    Project 1: Simple Calculator

Week 2: Control Flow and Loops

    Lesson 3: Conditional Statements
    Lesson 4: Loops in Python

    Project 2: Number Guessing Game

Week 3: Functions and Modular Code

    Lesson 5: Introduction to Functions
    Lesson 6: Modular Programming

    Project 3: Temperature Converter

Week 4: Data Structures: Lists and Dictionaries

    Lesson 7: Working with Lists
    Lesson 8: Dictionaries and Key-Value Pairs

    Project 4: Shopping List App

Week 5: Strings and File Handling

    Lesson 9: String Manipulation
    Lesson 10: File Input and Output

    Project 5: Note-Taking App

Week 6: Error Handling and Debugging

    Lesson 11: Error Handling in Python
    Lesson 12: Debugging Techniques

    Project 6: Number Validation Program

Week 7: Object-Oriented Programming (OOP)

    Lesson 13: Classes and Objects
    Lesson 14: Inheritance and Polymorphism

    Project 7: Animal Classification Program

Week 8: Final Project

    Lesson 15: Project Planning
    Lesson 16: Final Project Presentation

    Project 8: Choose Your Own Project (Game, To-Do List, etc.)
